Chapter 3. Transliterations

In general, transliteration IRIs are composed by joining via colons (1) the TAN namespace, (2) transliteration, (3) the language code that best fits the source alphabet, (4) the language code that best fits the target alphabet, (5) a name of the transliteration scheme. Because these elements are not predictable, especially the last one, this section will accumulate names for valid, relevant transliteration schemes.
